# Guess the Personality – Indian Cricketers (Vanilla JavaScript)

## Overview
This project is an image-based multiple-choice quiz game inspired by the
classic “Guess the Flag” concept.  
Users are shown a photograph of an Indian cricketer and must identify the
correct person from four given options.

The application is built using **pure (vanilla) HTML, CSS, and JavaScript**
with no external libraries or frameworks.

---

## Features
- Image-first quiz interface
- 4-option MCQ format
- 20 random questions per game
- Score increases on each correct answer
- Randomized options for every question
- Clean and responsive UI
- Stable Wikimedia image links

---

## Tech Stack
- **HTML5** – Structure
- **CSS3** – Styling and layout
- **Vanilla JavaScript** – Game logic and DOM manipulation
